# Dog fan

## サイト概要
### サイトテーマ

犬だけのSNSを作ることで犬好きの人が癒されるサイトを作りたいと思ったから

犬が好きな人

### 主な利用シーン
自分の買っている犬を投稿できる
犬のかわいいシーンを見ることができる
